Cycle Snapshot

The purpose of this page is to collect points of data from logged events then display those data points on charts and graphs so the user can visualize patterns throughout their cycle.

Data to be displayed:

  • BBT (number)
  • LH (number)
  • Flow (light, moderate, heavy)
  • Cramps (none, light, moderate, heavy)
  • Libido (low, moderate, high)
  • Mood (depressed, sad, numb, irritable, angry, sensitive, happy, excited, manic)
  • Energy level (exhausted, low, moderate, high, energized)
